# Customer Churn Analysis Dashboard

## Overview

This project analyzes customer churn data for a telecom company using **SQL Server** and **Power BI**. It demonstrates an end-to-end analytics workflow including data loading, cleaning, transformation, modeling, and interactive dashboard creation to identify customer churn patterns and business insights.

## Tech Stack

* SQL Server
* SQL
* Power BI
* Power Query (M)
* DAX

## Project Workflow

* Imported raw customer data into SQL Server.
* Performed data cleaning and preprocessing using SQL.
* Created production tables and SQL views for reporting.
* Connected Power BI to SQL Server.
* Built data transformations and calculated columns using Power Query.
* Created DAX measures for key business metrics.
* Developed an executive dashboard to analyze customer demographics, account information, geographic distribution, services, and churn trends.

## Dashboard Metrics

* Total Customers
* New Joiners
* Total Churn
* Churn Rate
* Customer Demographics
* Account & Contract Analysis
* Geographic Churn Distribution
* Service Usage Analysis

## Project Status

✅ SQL ETL Completed

✅ Executive Power BI Dashboard Completed
